#Windows Server 2013 provisioned with Vagrant, Puppet and Chocolatey

Provisioning boilerplate for Windows projects. Fork it and provision your stuff with ease.

###Preinstalled

- Git
- .Net4.5
- Visual Studio 2013 (takes 20min to provision)
- MsSQL
- Node.js
